Investigation Overview¶

In this project I will investigate the Ford GoBike System dataset
I will be looking for trends in the dataset and the majority of the users by what type the are and what gender
Key insights

  • The ride are usually between 3 - 12 minutes
  • Males are the majority of the users with 74.3%
  • Subscribers have higher ride duration

Dataset Overview¶

Dataset used Ford GoBike System Data provided by udacity
This data set includes information about individual rides made in a bike-sharing system covering the greater San Francisco Bay area
The dataset contains 83989 rows and 16 columns

(Visualization 1)¶

Fisrt Visualization it shows the duration of the bike rides in second and how many there are rides with that duration
With the plot we can see that the users mostly rent the bikes from 3 minutes to 12 minutes

(Visualization 2)¶

Second Visualization it shows the percentage of genders of the bike rides
With the plot we can see that our customers mostly are males with 74.3% and 23.6% for females

(Visualization 3)¶

Third Visualization it shows the duration in seconds of the bike riders based on the user type
